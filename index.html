REACT DLA ŚREDNIO ZAAWANSOWANYCH - MENTORZY.IT

React to technologia, która stała się wiodącą w świecie front-end developerów. Ta biblioteka JavaScript jest podstawą tworzenie wielu nowoczesnych aplikacji webowych. Kurs Daniela jest skierowany do osób, które podstawy mają już za sobą i chcą wejść poziom wyżej. Z kursu dowiesz się m.in. co to jest styled-components, Redux, React hooks, code-splitting czy Context API (ale i wielu innych rzeczy). Jeśli więc chcesz znacznie poszerzyć swoje horyzonty w React lub/i starasz się o pracę junior react developera, to ten kurs będzie dobrym wyborem.
